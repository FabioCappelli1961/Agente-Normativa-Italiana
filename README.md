# Agente-Normativa-Italiana

Agente autonomo AI per raccolta, analisi e sintesi automatica della normativa italiana.

## Descrizione

Monitora automaticamente:
- Codice Civile
- Normativa Fiscale
- Codice della Crisi d'Impresa
- Normativa Professionale
